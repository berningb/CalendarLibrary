Calendar Library

This is my calendar library. I wanted to add event abilities but that was actually a lot hard then I thought it would be. This library makes it simple toc reate a calendar and fill it with whatever kind of information you would like. I however, made it really easy to get the information for future and previous years. I found the date object really complicated and took it upon myself to make something a bit more firnedly. Enjoy :)

createCalendar

Creates a grid made up of a 7 X 6 to look like a calendar.

createTitle()

Creates a div on top of the calendar. THis div has two main divs inside which are the current day and the current month and year that you are looking at.

createFooter()

A simple div that contains your two main buttons 'next, and 'previous'.

createCalendar()

Creates a grid made up of a 7 X 6 to look like a calendar.

fillCalendar(date, month, year)

Fills the table or 'calendar' with the appropriete information. takes in date object, month, and year.

fillTitle(date, month, year)

appends and recieves all information for the title. Pass in a date, monht, and year.

fillFooter()

actually appends the buttons to the div

Calendar(id, date)

the basic constructor for the library. Takes in two params, an id and a date.

last()

Does the action of appending the last month's info to the Calendar.

next()

Does the action of appending the first month's info to the Calendar.

getCurrentYear()

simple method to get the current year. returns a number

getNextYear()

simple method to get the next year.Returns a number

getLastYear()

simple method to get the previous year. Returns a number

getNextMonth()

simple method to get the next month. returns number.

getCurrentMonth()

simple method to get the current month. Returns a number.

getLastMonth()

simple method to get the previous month. Returns a number
